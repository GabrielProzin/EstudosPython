# Roadmap Estruturado de Estudos – Machine Learning com Base Matemática

> **Carga diária:** ~1 hora  
> **Frequência:** todos os dias  
> **Foco:** Machine Learning  
> **Nível:** Python intermediário → ML consciente (sem superficialidade)

---

## Como usar este roadmap

- Este arquivo define **o que estudar e por quê**.
- Exercícios práticos ficam **exclusivamente** no arquivo:
  `Exercicios_Roadmap_Python_ML.md`
- Cada dia foi pensado para ~1h:
  - 40–45 min estudo conceitual
  - 15–20 min revisão/anotações
- Não avance se o conceito do dia **não estiver claro**.

## FASE 1 – Python para Machine Learning (Nível Intermediário)

> **Objetivo:** usar Python como ferramenta de raciocínio para ML, não como script básico.

### Semana 1 – Modelo mental correto em Python

**Dia 1 – Tipos, mutabilidade e estado**

- Referência vs cópia
- Bugs silenciosos causados por estado

**Dia 2 – Estruturas de dados em profundidade**

- list, tuple, set, dict
- Casos reais em dados
- Intuição de custo computacional

**Dia 3 – Funções como transformações**

- Funções puras vs efeitos colaterais
- Entrada → processamento → saída
- Funções como blocos de pipeline

**Dia 4 – Expressividade e padrões idiomáticos**

- Comprehensions
- enumerate, zip, any, all
- Leitura de código limpo

**Dia 5 – Organização e modularização**

- Separação de responsabilidades
- Estrutura mínima de projeto

**Dia 6 – Debug consciente**

- Leitura de traceback

**Dia 7 – Revisão**

- Conectar Python com problemas de ML
