O que é Machine Learning?
- É um campo de estudos onde o computador ganha a habilidade de aprender sem programar algo explicitamente.

Machine learning possui dois tipos de aprendizado:
- Aprendizado supervisionado (Supervised Learning): 
O modelo é treinando com um conjunto de dados que já possui "respostas" corretas (rótulos). Para cada (input), existe uma saída esperada (output).
Objetivo: É encontrar uma relação entre o input e o output.

Os dois tipos de aprendizados supervisionado é o de regressão e classificação.

- Regressão: O algoritmo de aprendizagem precisa prever números a partir de um número infinito de números de saída possíveis.

- Classificação: O algoritmo de aprendizagem precisa fazer uma previsão de categoria, de um pequeno conjunto de resultados possíveis.

- Aprendizado não supervisionado (Unsupervised Learning):
O modelo recebe dados sem respostas pré-definidas, não possui (rótulos) informando o que é certo ou errado.
Objetivo: A partir disso ele tenta identificar os padrões, estruturas e agrupamentos sem orientação externa.

O que é distribuição?

- Distribuição é como os dados estão espalhados.

Distribuição concentrada:
Os valores estão muito próximos uns dos outros, com baixa variabilidade.
Exemplo: [7, 7, 6.5, 7.2, 6.8, 7, 7.1]

Distribuição espalhada:
Os dados ocupam um intervalo grande, com valores bem distantes entre si.
Exemplo: [2, 4, 6, 7, 8, 9, 10]

Distribuição enviesada para baixo:
A maioria dos valores é alta, mas poucos valores muito baixos puxam a distribuição para a esquerda.
Exemplo: 1, 2, 2, 3, 4, 5, 9.
